# Hands-on: Create CLI With Cobra

## Step 01

```shell
cobra init --pkg-name todo
```

## Step 02

```shell
go mod init todo
```

## Step 03

```shell
 go build
```

## Step 04

```shell
./todo
```

## Step 05

```shell
cobra add hello
```

## Step 06

```shell
go build
```
